# CineVerse - Movie Review Website

A comprehensive movie review platform featuring in-depth analysis of films from various genres and languages, with a focus on Indian cinema including Hindi, Tamil, and Telugu movies.

## Features

- **Home Page**: Browse featured movies with ratings and brief descriptions
- **Top Rated Movies**: View top 5 movies in a ranked table format
- **Contact Page**: Get in touch with the CineVerse team
- **Responsive Design**: Optimized for desktop and mobile viewing
- **Movie Database**: Curated selection of popular films with expert reviews

## Movies Featured

- **Barfi** (Hindi) - Romance drama with a unique love story
- **Kabali** (Tamil) - Action thriller with intense sequences
- **RRR** (Telugu/Hindi) - Epic action film with spectacular stunts
- **Master** (Tamil) - Gripping action thriller
- **3 Idiots** (Hindi) - Thought-provoking drama about education and friendship
- And more...

## Technologies Used

- HTML5 for structure
- CSS3 for styling and responsive design
- Static website architecture

## How to Use

1. Clone the repository
2. Open `index.html` in your web browser
3. Navigate through the different sections using the menu
4. Click "Read Review" buttons to view detailed movie analysis
